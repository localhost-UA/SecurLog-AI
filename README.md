# SecurLog-AI
SecurLog Local AI 🛡️

Локальний AI-інструмент для аналізу системних журналів (Privacy First).

Цей проєкт призначений для системних адміністраторів та фахівців з кібербезпеки, які хочуть використовувати потужність LLM для аналізу логів, не ризикуючи конфіденційними даними компанії.
🚀 Чому це важливо?

Аналіз логів вручну займає години. Використання публічних сервісів (як ChatGPT) для аналізу внутрішніх логів — це ризик витоку корпоративних даних (IP-адрес, логінів, структури мережі).

SecurLog вирішує це:

    Анонімізація: Автоматично приховує чутливі дані (IP, MAC, логіни) перед аналізом.

    Локальний запуск: Весь аналіз виконується на вашому "залізі" через Ollama. Дані ніколи не покидають ваш комп'ютер.

    Візуалізація: Зручний веб-інтерфейс (Streamlit) для швидкого огляду загроз.

🛠 Технології

    Backend: Python

    LLM Engine: Ollama (локальний запуск моделей Phi-3/Llama-3)

    Interface: Streamlit

    Security: Regex-based sanitization

⚙️ Як запустити?

    Встановіть Ollama та запустіть модель: ollama run phi3

    Встановіть залежності: pip install requests rich streamlit pandas

    Запустіть проєкт: start.bat (або streamlit run app.py)
